# Data-Science-project
This project analyses a Kaggle dataset of AI-generated misinformation 
across four social media platforms (Facebook, Twitter, Reddit, and Telegram). 
Using R, it explores temporal patterns, geographical spread, and platform-level 
differences in misinformation distribution. Three machine learning models 
(Logistic Regression, Random Forest, and XGBoost) are trained and evaluated 
to predict whether a post contains misinformation.
